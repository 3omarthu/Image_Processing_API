### Scripts
- Install: ```npm install```
- Build: ```npm run build```
- Run unit tests: ```npm run test```
- Start server: ```npm run start```

### Usage
The server will listen on port 8080:

#### Brief instructions
http://localhost:8080/

#### Example 
http://localhost:8080/api/images?imagename=encenadaport&width=190&height=190
